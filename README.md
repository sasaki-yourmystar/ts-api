## Announce
```
npm install
```
してください

## Run ts

```
npm run ts-node index.ts
```

## ts -> js

```
npm run tsc
```

## Run js

```
node index.js
```

## execute API

```
% curl http://localhost:3000/test
->get

% curl -X POST http://localhost:3000/test
->post

% curl -X PUT http://localhost:3000/test
->put

% curl -X DELETE http://localhost:3000/test
->delete

```
